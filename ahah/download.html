To run Appreciating Hubble at Hyper-speed, first unzip the downloaded archive. Windows and Mac users, double-click ahah.jar. Linux users, run java -jar ahah.jar.

The Sun Java Runtime version 1.4.3 or newer is required to run Appreciating Hubble at Hyper-speed - the latest version can be downloaded from Java.com.

A configuration file is included that can be used to change the application's default behavior. To learn about the contents and format of this file, see the Configuration File section of the online manual.
